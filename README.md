## 🚀 Getting Started

### Prerequisites
- Java Development Kit (JDK) 17 or higher.
- A terminal or IDE of your choice.

### Running a Solution
1. Clone the repository:
   ```bash
   git clone https://github.com/krishnasahoo11156/DSA.git
   ```
2. Navigate to the desired folder:
   ```bash
   cd DSA/Basics
   ```
3. Compile and run:
   ```bash
   javac FileName.java
   java Basics.FileName
   ```

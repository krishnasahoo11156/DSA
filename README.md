<br/>

<details>
<summary><b>🧮 Calculator.java — Click to Expand</b></summary>
<br/>

> **Concept:** `if / else-if` conditionals | **Input:** Two integers + operator char

```java
// Takes two numbers and an operator (+, -, *, /) → prints the result
char op = sc.next().charAt(0);

if (op == '+') { System.out.println("Sum: " + (a + b)); }
else if (op == '-') { System.out.println("Difference: " + (b - a)); }
else if (op == '*') { System.out.println("Product: " + (a * b)); }
else if (op == '/') { System.out.println("Quotient: " + (a / b)); }
else { System.out.println("Invalid operator!"); }
```

**Sample Run:**
```
Enter the first number  : 12
Enter the second number : 4
Enter the operation (+  - / *) : *
The product of 12 and 4 is : 48
```

## 🚀 Getting Started

### Prerequisites

```bash
✔ Java Development Kit (JDK) 17 or higher  →  Recommended: JDK 25+
✔ VS Code (with Java Extension Pack)         →  or any Java-friendly IDE
✔ Git                                         →  for cloning
```

### ⚡ Clone & Run in 3 Steps

```bash
# 1️⃣  Clone the repository
git clone https://github.com/krishnasahoo11156/DSA.git
cd DSA

# 2️⃣  Compile any file  (example: CurrencyConverter.java)
javac first-java/CurrencyConverter.java

# 3️⃣  Run it!
java -cp first-java CurrencyConverter
```

> 💡 **Pro Tip:** Using VS Code? Just press `▶ Run` — the Java Extension Pack handles compile + run automatically!
